# 更新日志

## [Unreleased]

### 新增

- **系统托盘功能** - 应用现在显示在 Windows 通知区域（系统托盘），而不是任务栏
  - 点击托盘图标：显示/隐藏主窗口
  - 右键托盘图标：显示上下文菜单（显示窗口、隐藏窗口、退出）
  - 点击窗口关闭按钮：隐藏窗口而非退出应用
  - 任务栏不再显示应用图标
